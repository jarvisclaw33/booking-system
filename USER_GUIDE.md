# Booking System Dashboard - User Guide

## Getting Started

### Accessing the Dashboard

1. Navigate to the application URL
2. You'll be automatically redirected to either:
   - **Login Page** if you don't have an account
   - **Dashboard** if you're already logged in

### Account Creation

#### Signing Up
1. Click "Sign up" link on the login page
2. Enter your email address
3. Create a strong password
4. Confirm your password
5. Click "Sign Up"
6. Check your email for verification link
7. Click the verification link (if email verification is enabled)
8. Return to the app and log in with your credentials

#### Logging In
1. Enter your email address
2. Enter your password
3. Click "Sign In"
4. You'll be redirected to the dashboard

#### Logging Out
1. Click the "Logout" button in the sidebar (bottom)
2. You'll be redirected to the login page

## Dashboard Overview

The dashboard is divided into several sections accessible via the sidebar:

### Main Navigation (Sidebar)
- **Overview** - Dashboard home with key statistics
- **Locations** - Manage your booking locations
- **Bookings** - View and manage all bookings
- **Calendar** - Visual calendar view of bookings

## Features

### 1. Overview Dashboard

The overview page displays key metrics about your booking system:

#### Statistics Cards
- **Total Bookings** - All bookings in the system
- **Locations** - Number of locations you manage
- **Upcoming** - Bookings scheduled for the future
- **Average per Location** - Average bookings per location

#### What It Shows
- Real-time statistics
- Quick health check of your business
- Upcoming bookings count for planning

#### How to Use
- Check statistics to understand your booking patterns
- Use upcoming bookings count to manage capacity
- Monitor location performance

---

### 2. Locations Management

Manage all your booking locations in one place.

#### Creating a Location

1. Click the **"Add Location"** button (top right)
2. Fill in the location details:
   - **Location Name** (required) - Name of your location
   - **Address** (optional) - Full address for customer reference
   - **Timezone** (optional) - Select your location's timezone
3. Click **"Save Location"**

#### Location Details
Each location card shows:
- Location name
- Address (if provided)
- Timezone
- Edit and delete buttons

#### Managing Locations

##### Viewing Locations
- Locations are displayed in a grid format
- Scroll to view all your locations
- Use responsive design that adapts to your screen size

##### Deleting a Location
1. Click the trash icon on a location card
2. Confirm the deletion in the dialog
3. Location will be immediately removed

##### Empty State
If you have no locations:
- A helpful message appears
- Quick "Create Location" button
- Encourages you to start

#### Best Practices
- Use clear, descriptive location names
- Include full addresses for customer clarity
- Select the correct timezone for accurate booking times
- Create locations before creating bookings

---

### 3. Bookings Management

View and manage all customer bookings.

#### Creating a Booking

1. Click the **"New Booking"** button (top right)
2. Fill in booking details:
   - **Guest Name** (required) - Customer name
   - **Location** (required) - Select from your locations
   - **Start Time** (required) - When the booking begins
   - **End Time** (required) - When the booking ends
   - **Status** (optional) - Pending, Confirmed, or Cancelled
3. Click **"Save Booking"**

#### Viewing Bookings

Bookings are displayed as cards with:
- Guest name
- Location
- Date and time
- Status badge with color coding:
  - **Green** = Confirmed
  - **Yellow** = Pending
  - **Red** = Cancelled

#### Filtering Bookings

Use the status filter dropdown to view:
- **All Bookings** - Show all bookings regardless of status
- **Confirmed** - Only confirmed bookings
- **Pending** - Only pending bookings
- **Cancelled** - Only cancelled bookings

#### Managing Bookings

##### Editing a Booking
1. Click the edit icon on a booking card
2. Update the details as needed
3. Click "Save" to apply changes

##### Deleting a Booking
1. Click the trash icon on a booking card
2. Confirm the deletion
3. Booking will be removed from the system

##### Status Management
- Change booking status from Pending to Confirmed
- Mark bookings as Cancelled as needed
- Filter to see bookings by status

#### Empty State
If you have no bookings:
- A helpful message appears
- Quick "New Booking" button
- Encourages you to create your first booking

#### Tips & Tricks
- Ensure start time is before end time
- Select the correct location
- Update status promptly as bookings progress
- Use filters to find specific bookings quickly

---

### 4. Calendar View

Visual calendar representation of your bookings.

#### Calendar Features
- **Month Navigation**:
  - Previous month button (left arrow)
  - Next month button (right arrow)
  - "Today" button to jump to current date
  
- **Calendar Grid**:
  - Sunday through Saturday layout
  - Current month dates in white
  - Previous/next month dates in gray
  - Today's date highlighted in blue

#### Viewing Bookings on Calendar
- Booking indicators appear on dates with bookings
- Shows count of bookings per day
- Color-coded to indicate booking status
- Click on a date to see bookings for that day

#### Using the Calendar
1. Navigate to Calendar section from sidebar
2. Choose month using navigation buttons
3. View bookings displayed on dates
4. Click "Today" to return to current month
5. Identify busy days and plan accordingly

#### Calendar Tips
- Plan capacity based on busy days
- Identify gaps for marketing or promotions
- Monitor peak booking periods
- Use to communicate availability to customers

## Common Tasks

### Task: Set Up Your First Booking

1. **Create a Location**
   - Go to Locations section
   - Click "Add Location"
   - Enter: Name, Address, Timezone
   - Save

2. **Create a Booking**
   - Go to Bookings section
   - Click "New Booking"
   - Select the location you created
   - Enter guest name and time
   - Set status to "Confirmed"
   - Save

3. **View on Calendar**
   - Go to Calendar section
   - Navigate to the month of your booking
   - See the booking appear on the calendar

### Task: Manage Multiple Locations

1. Create each location with unique name
2. Create bookings specifying which location
3. Use filters to view bookings by location
4. Monitor statistics by location type

### Task: Handle Booking Requests

1. **New Request**:
   - Create booking with "Pending" status
   
2. **Request Confirmed**:
   - Edit booking
   - Change status to "Confirmed"
   - Save

3. **Request Cancelled**:
   - Edit booking
   - Change status to "Cancelled"
   - Save

### Task: Monitor Availability

1. Go to Calendar
2. Identify open dates and times
3. Check statistics for capacity planning
4. Locations page shows all available spots
5. Create new bookings for available slots

## Troubleshooting

### I Can't Log In
- Verify email address is correct
- Check that password is entered correctly
- Ensure Caps Lock is off
- Try resetting your password

### Bookings Not Appearing
- Verify location is selected
- Check that times are valid (start before end)
- Ensure you're looking at correct month in calendar
- Try refreshing the page

### Missing Locations
- Check if locations were saved
- Verify you're logged into correct account
- Try navigating away and back
- Refresh the page if needed

### Booking Time Issues
- Verify timezone is correct in location
- Check that you're entering times in local timezone
- Confirm no conflicts with existing bookings
- Double-check time format (24-hour or 12-hour)
